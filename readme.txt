Perfect Coming Soon page enables you to use a light weighted plugin for multiple needs of coming soon,underconstruction or offline mode.

== Description ==

Perfect coming soon page allows user to enable easy maintenance, coming soon or under construction mode without any code knowledge. 
Plugin allows only logged in (Administrator) to view website. Rest of visitors are landed on maintenace, coming soon or underconsturction mode page.


### Plugin Features & Options:

*   Responsive Coming soon page for all devices
*   Cross Browser compatible
*   Customize the options and settings for title, text & content to display
*   Setup your social media profiles
*   One Click Installation & NO Coding Knowledge is required.
*   Light Weighted & Clean code that makes page load fast 
*   Modern Design & Clean UI look 
 
== Installation ==

= Using WordPress Dashboard =

1. Navigate to 'Add New' in the plugins dashboard
2. Search for 'Perfect Coming Soon Page'
3. Click 'Install Now' Button
4. Activate plugin & its done

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `perfect-coming-soon-page.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `perfect-coming-soon-page.zip`
2. Extract the `perfect-coming-soon-page` directory to your computer
3. Upload the `perfect-coming-soon-page` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin's dashboard


== Frequently Asked Questions ==
= How can i activate the maintenance/coming soon/offline mode ? =

Once after the Installation & Activation of plugin, automatically coming soon page is activated. Now you can move to Settings >> Perfect Coming Soon Page to customize content and social media links. 
 
= Can i upload custom backgroud image ? =
Yes, Upload image in media library or copy url of image and paste in settings.

= Disabled Plugin and it still shows =

This generally happens when you are using a cache plugin. Kindly clear cache and hit refresh button.
